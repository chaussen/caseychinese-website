# Images

Drop your photos into this folder using the same filenames listed below.
GitHub Pages will serve them automatically from your custom domain.

| Filename | Where it appears | Suggested size |
|---|---|---|
| hero.jpg | Hero portrait, right side of homepage | 800 × 1000 (portrait 4:5) |
| gallery-1.jpg | Gallery, large feature tile | 1000 × 700 |
| gallery-2.jpg | Gallery, second large tile | 800 × 700 |
| gallery-3.jpg | Gallery, small tile | 600 × 400 |
| gallery-4.jpg | Gallery, small tile | 600 × 400 |
| gallery-5.jpg | Gallery, wide tile | 800 × 400 |
| gallery-6.jpg | Gallery, wide tile | 800 × 400 |
| gallery-7.jpg | Gallery, wide tile | 800 × 400 |

JPG, PNG, and WebP all work. Keep individual files under ~500 KB for fast page loads.
The current files are SVG placeholders — replace them by uploading a real photo with the same filename (e.g. `hero.jpg`).
